    Annew.AutofacProfiling

    This is a really tiny library that lets you profile Autofac resolutions in MiniProfiler. In short - every time Autofac resolves a component, you get a line in MiniProfiler telling you how long it took. Very handy if you're having trouble tracking down performance issues in apps.

    To get started:

    1. Install Annew.AutofacProfiling via the NuGet package manager console.
    2. Add a call to EnableProfiling() when you build your container, as below:
    var builder = new ContainerBuilder();
// Your registrations go here.
var container = builder.Build();
// <magic>
container.EnableProfiling();
//</magic>
DependencyResolver.SetResolver(new AutofacDependencyResolver(container));

    ... and that's it. So long as you've set up MiniProfiler, you'll now get profiling for each component that Autofac resolves, even if it's buried deep down in the object graph.

    Dependencies:

    • Autofac
    • MiniProfiler
